
# Prisma

Prisma is a custom Minecraft server written in C# to help with exploring networking, client-server relationships, and game design at a more technical level. In it's current state it is definitely not fit for anything other than tinkering and should not be used in a production environment.

![Repo Stars](https://img.shields.io/github/stars/iffyspeak-chroma/Prisma?style=flat-square)



## Installation

Clone the project

```bash
git clone https://github.com/iffyspeak-chroma/Prisma.git
cd Prisma
```

Then open in your favorite C# IDE and build as usual.

## Contributing

I fully support contributing to the project but I cannot guarentee your contribution will be merged into the project. There's not exactly a code of conduct for contributing or anything but general rules are as follows:
- No intentionally harmful contributions
- No README changes only to become a contributor (if you actually contribute something: you may sign your name)
- Ensure contributions are "functional" (they don't have to work fully, just as long as they do something without causing a crash, it's fairly functional)
